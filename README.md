## Как задеплоить

1. Загрузи **всё содержимое этой папки** на GitHub
2. Зайди в **Actions** → запусти **"Download Hero Images"** (кнопка "Run workflow")
3. GitHub сам скачает все 127 фото героев в папку `heroes/`
4. Включи **Pages** в Settings → Pages → Source: main branch
5. Готово — сайт работает с фото!

> Фото скачиваются автоматически при каждом push, повторно запускать не нужно.
